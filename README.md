# Smart Campus Navigation

## Overview
A mobile application that helps students navigate university campus buildings with indoor floor maps, AR-assisted wayfinding, and real-time room availability integration with the university timetable system.

## Tech Stack
- React Native
- Firebase
- Google Maps API
- Node.js

## Architecture
- **Frontend**: React Native
- **Backend**: Node.js (Express)
- **Database**: Firebase

## Quick Start

### Prerequisites
- Node.js 18+



### Installation
```bash
# Frontend
cd frontend && npm install

# Backend
cd backend && npm install

# Environment
cp backend/.env.example backend/.env
# Edit .env with your credentials

# Database setup



# Run
cd backend && npm run dev
# In another terminal:
cd frontend && npm run dev
```
